Add seats#show route and action

Render the contents of the seats#show controller action into the same application frame as seats#index, but embedded within a block wrapped by a browser-native <dialog> element. Specify the <dialog> element with an open attribute.

On turbolinks:load, find <dialog> elements with the open attribute and manually invoke dialog.showModal() to render the dialog modally.
